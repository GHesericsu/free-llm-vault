# Free LLM Vault

Collaborative research and configuration for routing free LLM APIs.

## Project Goal
Maximize free LLM quota by intelligently routing requests across multiple providers like Groq, Cerebras, and Mistral using LiteLLM.

## Directory Structure
- `/docs`: Architecture and provider research.
- `/providers`: Step-by-step signup guides.
- `/routing-configs`: LiteLLM and OpenClaw configuration templates.
